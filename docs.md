```r
$ python setup.py
```
than copy JDB.ini in /data/data/com.termux/files/usr/etc/JDB/JDB
ini
than copy JDB and jdb to /data/data/com.termux/files/usr/bin/
to start database enter in terminal
```r
$ JDB
```
to connect 
```r
$ jdb -p password
```
